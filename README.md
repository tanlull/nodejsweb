Node JS trainning @TOT by Coach Ake (Akenarin Komkoon)

**Setup**
1. Install & Start Mysql
2. Create database & populate data by running "createDB.sql"
3. Config database connection at : "config.js"
 

**To Run** 
```javascript
npm start
```

**Then open browser at**  

```
http://localhost:3000
```
